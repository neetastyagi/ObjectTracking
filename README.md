## Problem Set 5: Object Tracking and Pedestrian Detection

### Description

This project implements tracking methods for image sequences and videos using computer vision techniques. The main algorithms implemented are the **Kalman Filter** and **Particle Filter** for robust object tracking and pedestrian detection in video sequences.

### Project Overview

This is a **Python-based** project that focuses on:
- Object tracking across image sequences
- Pedestrian detection in video data
- Implementation of Kalman filtering for motion estimation
- Implementation of Particle filtering for non-linear state estimation
- Integration of tracking methods for real-time video analysis

### Key Features

- **Kalman Filter**: For linear, Gaussian state estimation and tracking
- **Particle Filter**: For non-linear and non-Gaussian filtering scenarios
- **Object Detection**: Methods for identifying and locating pedestrians in video frames
- **State Prediction**: Predicting object trajectories across frames
- **Multi-object Tracking**: Handling multiple objects in the same scene

### Technologies Used

- **Language**: Python
- **Purpose**: Computer Vision and Object Tracking
